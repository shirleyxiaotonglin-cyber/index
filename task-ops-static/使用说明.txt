Task Ops 离线版 — 直接使用
============================

1. 用浏览器打开本文件夹里的 index.html
   · 可以双击 index.html，或把 index.html 拖到浏览器窗口里
   · 推荐使用 Chrome / Edge / Safari

2. 无需安装 Node.js，无需联网（界面与数据都在本机）

3. 演示登录
   邮箱：admin@demo.com
   密码：demo123
   （manager@demo.com、member@demo.com 密码相同）

4. 数据保存在当前浏览器的「本地存储」中；清除站点数据会重置演示数据

5. 跨设备同步需要「页面」与「/api 后端」在同一站点或由你手动指定 API 地址：
   · 推荐：在 Vercel 只部署仓库里的 task-ops（Next.js），数据库迁移 + AUTH_SECRET 配好，
     用 https://你的域名/workgraph.html 打开协作页（同域 /api，手机与电脑数据一致）。
   · 若只部署纯静态 task-ops-static、API 在另一域名，请在控制台执行：
     localStorage.setItem("workgraph_api_base","https://你的后端根地址")
     后刷新（不要末尾斜杠）。

6. 若双击后仍是空白页，请右键 index.html → 打开方式 → 选浏览器

7. 若数据无法保存，请换用 Chrome 或 Edge 打开；不要用「无痕/隐私」模式
